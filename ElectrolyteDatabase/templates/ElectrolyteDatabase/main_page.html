Help

How are Electrodes organized in our database?
In our database, the composition of a cathode/anode is described in terms of a specific ratio of its components
(ie: NMC532 or NMC622. ) Each of these specific ratios are organized in terms of their family; for example, the
family that ‘NMC532’ belongs to is ‘NMC.’

Searching for Wet Cells
To use our tool to search wet cells, press 'Specify and Edit Wet Cells' on the main page. From there, you will need to fill out
the fields to specify both a list of boxes and a list of electrolytes. Specify as much information as you can in order to get more
accurate search results. After you have specified these two lists, click 'Update Cell Queryset' to generate a list of wet cells that exclusively
uses boxes and electrolytes from the previously generated lists.
After generating the list of wet cells, if you wish to edit the saved data associated with the cells, click 'edit metadata for cell queryset'.


Modifying and Adding Dry Cells to the Database
This web app gives lab members the ability to add new dry cells into the database as well as modify the data related to existing
dry cell. To use either of these tools, click 'Register or Modify Dry Cell Models' on the main page. From there, if you want to edit an existing dry cell,
select the model in question in the drop down and click 'Choose Cell.' To create a new model, click 'Register New Dry Cell.'

If you choose to edit a dry cell, every database field relating to these cells will be generated, and they will be populated wioth
the initial data that is already in the database. To edit this information, simply replace any of the data within these fields with the updated information, and
then click 'Update Dry Cell.' If you are creating a new dry cell, the process is the same however all of the fields will be initially empty.
